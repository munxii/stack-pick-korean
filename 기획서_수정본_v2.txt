프로젝트 전체 정리 (수정본 v2)
====================================

1. 프로젝트 개요
────────────────
프로젝트명

개발자·크리에이터용 SaaS 요금제 비교 및 구독 최적화 플랫폼
가칭:
  PlanPick / StackPlan / Toolly / PriceStack / SubPick

한 줄 소개

AI 툴, 호스팅, 데이터베이스, 협업 툴 등 다양한 온라인 서비스의 무료/유료 요금제를 한눈에
비교하고, 사용자에게 가장 적합한 구독 조합을 추천해주는 플랫폼

핵심 아이디어

개발자나 학생, 스타트업, 크리에이터들은 프로젝트를 진행할 때 여러 온라인 서비스를 함께
사용한다. 예를 들면: ChatGPT, Claude, Vercel, Supabase, Notion, GitHub, Figma 등.

이런 서비스들은 각각 무료 요금제와 유료 요금제가 다르고, 제한 조건도 다르다.
하지만 이 정보들은 각 공식 사이트에 흩어져 있어 비교가 어렵다.

그래서 이 플랫폼은:
  - 카테고리별 서비스 탐색
  - 무료 플랜 여부 확인
  - 유료 요금제 비교
  - 기능 차이 비교
  - 사용자 상황에 맞는 최적 조합 추천
  - 여러 서비스 구독 시 비용 절감 분석
까지 제공하는 비교 + 추천 + 관리 플랫폼이 되는 것이 목표다.


2. 문제 정의
────────────
기존 문제점

1) 정보가 흩어져 있음
   서비스마다 공식 사이트에 들어가야 하고, 가격 정책 페이지 구조도 다 다르다.

2) 무료 플랜 한도가 직관적이지 않음
   "무료 제공"이라고 해도 실제로 어디까지 가능한지 명확하지 않은 경우가 많다.

3) 여러 서비스 조합의 총비용을 알기 어려움
   한 개 서비스 가격은 알아도, 여러 개를 같이 쓸 때 월 총액이 얼마인지 계산하기 번거롭다.

4) 비슷한 기능을 하는 서비스끼리 비교가 어렵다
   예: ChatGPT vs Claude / Vercel vs Netlify / Supabase vs Firebase / Notion vs Coda

5) 학생/개인 개발자/팀마다 필요한 조건이 다르다
   누군가는 무료 플랜이 중요하고, 누군가는 협업 기능이 중요하며,
   누군가는 API 제공 여부가 중요하다.

6) 요금제 정보가 수시로 바뀐다  ★ 신규
   SaaS 서비스들은 요금제를 자주 변경하며, 사용자가 이를 실시간으로 파악하기 어렵다.
   (예: Notion 2023년 무료 플랜 축소, Heroku 무료 플랜 폐지 등)


3. 서비스 목표
────────────────
세부 목표
  - 다양한 온라인 서비스의 요금제 정보를 통합 제공
  - 무료 플랜과 유료 플랜의 차이를 명확히 시각화
  - 카테고리별 서비스 탐색 지원
  - 여러 서비스를 동시에 비교 가능하게 구성
  - 예산과 목적에 맞는 추천 기능 제공
  - 사용자의 구독 비용을 절감할 수 있는 조합 제안
  - 요금제 변경 감지 및 알림 시스템 운영  ★ 신규
  - 장기적으로는 제휴 할인 및 구독 관리 기능까지 확장


4. 타겟 사용자
────────────────
주요 타겟
  1) 대학생 / 과제 사용자
     무료 플랜 위주로 툴을 찾는 사용자
     웹사이트 제작, 팀플, 발표자료 제작 등에 필요한 툴을 찾는 경우

  2) 개인 개발자 / 사이드 프로젝트 사용자
     최소 비용으로 서비스 조합을 구성하려는 사용자
     호스팅, DB, AI 툴 등을 함께 비교하려는 경우

  3) 스타트업 / 초기 창업 팀
     여러 SaaS 툴을 도입해야 하는데 비용 최적화가 필요한 팀
     팀 단위 협업 요금제와 개인 요금제를 비교해야 하는 사용자

  4) 크리에이터 / 디자이너 / 마케터
     AI 툴, 협업 툴, 디자인 툴, 생산성 툴을 함께 사용하는 사용자

  └─────────────────┴──────────────────────────────┴────────────────────────────┘

우리 서비스의 차별점
  - 위 서비스들은 "어떤 서비스가 있나" 수준에서 멈춤
  - 이 플랫폼은 여러 서비스 조합의 총비용 계산 + 더 저렴한 대체 조합 추천까지 제공
  - 요금제 변경 감지 및 알림 기능으로 항상 최신 정보 유지
  - 개인/팀 규모별 맞춤 추천이라는 점에서 차별화


6. 서비스 핵심 가치
────────────────────
  1) 비교    - 흩어진 요금제 정보를 한곳에 모아 비교
  2) 시각화  - 무료/유료/추천 여부를 배지, 카드, 태그, 표로 직관적으로 확인
  3) 추천    - 사용자의 예산, 목적, 사용 서비스 개수에 맞는 조합 추천
  4) 절약    - 단순 정보 제공이 아닌 실제로 더 저렴한 조합 탐색 지원
  5) 최신성  - 요금제 변경 감지 시스템으로 항상 정확한 정보 제공  ★ 신규
  6) 관리    - 현재 쓰는 구독 서비스들을 저장하고 관리하는 기능 (장기)


7. 데이터 최신성 유지 전략  ★ 신규 섹션
────────────────────────────────────────
문제

SaaS 요금제 정보는 수시로 바뀐다. 정보가 오래되면 플랫폼의 신뢰성이 떨어진다.

대응 전략

  1단계 (MVP): 수동 관리 + 사용자 제보
    - 서비스 데이터는 관리자가 직접 입력/수정
    - "이 정보가 잘못되었나요?" 신고 버튼 제공
    - 각 서비스 카드에 "마지막 업데이트 날짜" 표시
    - 공식 사이트 링크 제공으로 사용자가 직접 확인 가능

  2단계 (성장기): 반자동화
    - Python 기반 웹 크롤러로 주요 서비스 가격 페이지 주기적 모니터링
    - 변경 감지 시 관리자 알림 → 검토 후 반영
    - Pro/Team 사용자에게 요금제 변경 알림 발송

  3단계 (장기): 자동화 + 커뮤니티
    - 공식 API 연동 (제공하는 서비스에 한해)
    - 커뮤니티 기여 시스템: 사용자가 직접 정보 업데이트 제안
    - 변경 이력 관리로 요금제 히스토리 확인 가능

신뢰도 표시 방식
  - 🟢 최근 30일 이내 확인됨
  - 🟡 30~90일 경과 (확인 권장)
  - 🔴 90일 초과 (공식 사이트 확인 필요)

11. 사이트 자체 요금제
──────────────────────
  1) Free   - 기본 탐색, 서비스 조회, 기본 검색, 2개 서비스 비교
  2) Plus   - 5개 비교, 고급 필터, 북마크, 가격 변동 알림, 개인 맞춤 추천
  3) Pro    - 무제한 비교, 조합 총 비용 계산, 절약 리포트, 대체 툴 추천, 할인 정보 우선 제공
  4) Team   - 팀 워크스페이스, 좌석 수 기반 비용 계산, 팀 단위 추천, 협업 대시보드

Pro 플랜 킬러 기능 (전환율 제고)  ★ 강화
  - 📊 "절약 리포트": 현재 조합 대비 절감 가능 금액을 PDF로 다운로드
  - 🔔 "가격 변동 알림": 사용 중인 서비스 요금 변경 시 즉시 이메일/알림
  - 🔀 "스마트 대체 추천": 비슷한 기능의 더 저렴한 서비스 자동 추천
  - 📅 "요금제 히스토리": 서비스별 가격 변동 이력 조회
  - 🏷️ "할인/프로모션 정보": 학생 할인, 스타트업 크레딧 등 우선 제공

──────────────────────────
시나리오 1. 대학생

  → ChatGPT/Claude/Copilot 비교 → Vercel/Render 비교 → 조합 생성 → 총 월 비용 확인

시나리오 3. 스타트업 팀
  "팀 전체가 쓸 협업/개발 툴을 정하고 예산을 맞추고 싶다."
  → Slack/Notion/Jira 비교 → 팀 규모 입력 → 좌석 수 기반 예산 계산 → Team 플랜 리포트

17. 프로젝트의 핵심 한 줄 (발표용)
──────────────────────────────────
버전 1: 다양한 온라인 도구의 요금제를 비교하고, 가장 합리적인 구독 조합을 추천하는 플랫폼
버전 2: 개발자와 팀을 위한 SaaS 가격 비교 및 구독 최적화 서비스
버전 3: 무료 플랜부터 유료 플랜까지, 온라인 도구 선택을 더 쉽고 똑똑하게 만드는 플랫폼
버전 4: 여러 SaaS를 함께 쓸수록 더 큰 가치를 제공하는 구독 최적화 플랫폼


18. 발표용 핵심 포인트
──────────────────────
  문제:    온라인 도구의 가격 정보가 흩어져 있고, 비교하기 어려우며, 수시로 바뀐다.
  해결:    카테고리별 서비스를 모아 무료/유료 요금제를 한눈에 비교할 수 있게 한다.
  차별성:  여러 서비스를 묶어서 전체 비용 계산 + 더 저렴한 조합 추천까지 제공한다.
  신뢰성:  요금제 변경 감지 시스템으로 항상 최신 정보를 유지한다.  ★ 신규
  확장성:  비교 사이트 → 구독 관리 → 가격 알림 → 팀 플랜 추천까지 단계적 확장 가능.
