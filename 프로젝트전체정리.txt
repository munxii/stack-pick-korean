프로젝트 전체 정리
1. 프로젝트 개요
프로젝트명

개발자·크리에이터용 SaaS 요금제 비교 및 구독 최적화 플랫폼
가칭:

PlanPick
StackPlan
Toolly
PriceStack
SubPick
한 줄 소개

AI 툴, 호스팅, 데이터베이스, 협업 툴 등 다양한 온라인 서비스의 무료/유료 요금제를 한눈에 비교하고, 사용자에게 가장 적합한 구독 조합을 추천해주는 플랫폼

핵심 아이디어

개발자나 학생, 스타트업, 크리에이터들은 프로젝트를 진행할 때 여러 온라인 서비스를 함께 사용한다.
예를 들면:

ChatGPT
Claude
Vercel
Supabase
Notion
GitHub
Figma

이런 서비스들은 각각 무료 요금제와 유료 요금제가 다르고, 제한 조건도 다르다.
하지만 이 정보들은 각 공식 사이트에 흩어져 있어 비교가 어렵다.

그래서 이 플랫폼은:

카테고리별 서비스 탐색
무료 플랜 여부 확인
유료 요금제 비교
기능 차이 비교
사용자 상황에 맞는 최적 조합 추천
여러 서비스 구독 시 비용 절감 분석

까지 제공하는 비교 + 추천 + 관리 플랫폼이 되는 것이 목표다.

2. 문제 정의
기존 문제점

현재 사용자들은 다음과 같은 불편을 겪는다.

1) 정보가 흩어져 있음

서비스마다 공식 사이트에 들어가야 하고, 가격 정책 페이지 구조도 다 다르다.

2) 무료 플랜 한도가 직관적이지 않음

“무료 제공”이라고 해도 실제로 어디까지 가능한지 명확하지 않은 경우가 많다.

3) 여러 서비스 조합의 총비용을 알기 어려움

한 개 서비스 가격은 알아도,
여러 개를 같이 쓸 때 월 총액이 얼마인지 계산하기 번거롭다.

4) 비슷한 기능을 하는 서비스끼리 비교가 어렵다

예:

ChatGPT vs Claude
Vercel vs Netlify
Supabase vs Firebase
Notion vs Coda
5) 학생/개인 개발자/팀마다 필요한 조건이 다르다

누군가는 무료 플랜이 중요하고,
누군가는 협업 기능이 중요하며,
누군가는 API 제공 여부가 중요하다.

3. 서비스 목표
목표

이 플랫폼은 사용자가 자신의 상황에 맞는 온라인 도구를 더 빠르고 합리적으로 선택하도록 돕는 것을 목표로 한다.

세부 목표
다양한 온라인 서비스의 요금제 정보를 통합 제공
무료 플랜과 유료 플랜의 차이를 명확히 시각화
카테고리별 서비스 탐색 지원
여러 서비스를 동시에 비교 가능하게 구성
예산과 목적에 맞는 추천 기능 제공
사용자의 구독 비용을 절감할 수 있는 조합 제안
장기적으로는 제휴 할인 및 구독 관리 기능까지 확장
4. 타겟 사용자
주요 타겟
1) 대학생 / 과제 사용자
무료 플랜 위주로 툴을 찾는 사용자
웹사이트 제작, 팀플, 발표자료 제작 등에 필요한 툴을 찾는 경우
2) 개인 개발자 / 사이드 프로젝트 사용자
최소 비용으로 서비스 조합을 구성하려는 사용자
호스팅, DB, AI 툴 등을 함께 비교하려는 경우
3) 스타트업 / 초기 창업 팀
여러 SaaS 툴을 도입해야 하는데 비용 최적화가 필요한 팀
팀 단위 협업 요금제와 개인 요금제를 비교해야 하는 사용자
4) 크리에이터 / 디자이너 / 마케터
AI 툴, 협업 툴, 디자인 툴, 생산성 툴을 함께 사용하는 사용자
5. 서비스 핵심 가치
1) 비교

흩어진 요금제 정보를 한곳에 모아 비교할 수 있음

2) 시각화

무료/유료/추천 여부를 배지, 카드, 태그, 표로 직관적으로 확인 가능

3) 추천

사용자의 예산, 목적, 사용 서비스 개수에 맞는 조합 추천 가능

4) 절약

단순 정보 제공이 아니라, 실제로 더 저렴한 조합을 찾도록 도와줌

5) 관리

장기적으로는 현재 쓰는 구독 서비스들을 저장하고 관리하는 기능까지 확장 가능

8. 차별화 기능

여기서부터가 네 프로젝트의 핵심 차별점임.

1) 여러 서비스 조합 기준 비교

기존 비교 사이트는 보통 서비스 하나하나를 개별적으로 보여준다.
하지만 이 플랫폼은 여러 서비스를 조합해서 전체 비용과 효율을 분석할 수 있다.

예:

ChatGPT + Vercel + Supabase + Notion
Claude + Render + Firebase + Slack

이런 식으로 조합을 구성하고 총 비용을 계산 가능

2) 비용 절감 중심 추천

단순히 “어떤 서비스가 있나”를 보여주는 것이 아니라,
같은 목적을 더 저렴하게 달성할 수 있는 조합을 추천한다.

예:

현재 조합 월 48달러
대체 조합 월 31달러
예상 절감액 월 17달러
3) 사이트 자체 요금제 도입

이 플랫폼도 자체 요금제를 두고,
고급 비교·추천·관리 기능을 제공한다.

10. “사이트 개수별 할인” 기능 설계

이건 네 서비스만의 특징으로 강조 가능함.

개념

사용자가 이 플랫폼에서 여러 구독 서비스를 관리하거나 비교할수록,
더 많은 최적화 기능과 혜택을 제공하는 구조

표현 방식

직접 “사이트 수가 많으면 할인”보다는,
다음처럼 표현하는 게 더 자연스럽고 서비스다움.

표현 예시
여러 SaaS를 함께 비교할수록 더 큰 절약 추천 혜택 제공
3개 이상의 툴을 사용하는 사용자에게 맞춘 번들 최적화 플랜
여러 서비스를 쓰는 팀을 위한 통합 구독 관리 플랜
다양한 구독 조합을 가장 합리적으로 설계할 수 있는 Pro 플랜
실제 기능 방식 예시
방식 1. 비교 확장형
Free: 2개 비교
Plus: 5개 비교
Pro: 무제한 비교
방식 2. 조합 최적화형

사용자가 여러 서비스를 선택하면:

월 총 비용 계산
중복 기능 탐지
더 저렴한 대체 툴 추천
가장 효율적인 조합 제시
방식 3. 절약 리포트형
현재 조합 예상 월 비용

절감 가능 금액
추천 대안 조합
방식 4. 할인/제휴 연계형

장기적으로:

15. 프로젝트 차별성 요약
기존 서비스와의 차이

기존 사이트들은 보통:

무료 서비스 목록만 보여주거나
서비스 대안만 추천하거나
리뷰만 제공함

하지만 이 프로젝트는:

서비스 카테고리화
무료/유료 요금제 시각 비교
여러 서비스 조합 분석
사용자 맞춤 추천
사이트 자체 요금제를 통한 구독 관리 기능

까지 포함하는 비교 + 추천 + 절약 플랫폼이라는 점에서 차별화됨

16. 프로젝트의 핵심 한 줄

발표용으로 좋은 문장들 적어줄게.

버전 1

다양한 온라인 도구의 요금제를 비교하고, 가장 합리적인 구독 조합을 추천하는 플랫폼

버전 2

개발자와 팀을 위한 SaaS 가격 비교 및 구독 최적화 서비스

버전 3

무료 플랜부터 유료 플랜까지, 온라인 도구 선택을 더 쉽고 똑똑하게 만드는 플랫폼

버전 4

여러 SaaS를 함께 쓸수록 더 큰 가치를 제공하는 구독 최적화 플랫폼

17. 발표용 핵심 포인트

교수님이나 팀원 앞에서 설명할 때는 이렇게 가면 됨.

문제

온라인 도구의 가격 정보가 흩어져 있고 비교가 어렵다.

해결

카테고리별로 서비스를 모아 무료/유료 요금제를 한눈에 비교할 수 있게 한다.

차별성

여러 서비스를 묶어서 전체 비용을 계산하고, 더 저렴한 조합까지 추천한다.

확장성

단순 비교 사이트를 넘어 구독 관리, 가격 변동 알림, 팀 플랜 추천까지 확장 가능하다.
